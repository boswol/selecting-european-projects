# Funding Portals by Country / Programme

Use this reference when deciding which portals to search in `/find-calls`. Prioritise the portals most relevant to the research profile (EU-wide first, then national).

---

## EU-wide / Supranational

| Portal | URL | Notes |
|---|---|---|
| EU Funding & Tenders Portal | https://ec.europa.eu/info/funding-tenders/opportunities/portal | Central portal for all EU programmes incl. Horizon Europe, LIFE, EIC |
| Horizon Europe (programme page) | https://research-and-innovation.ec.europa.eu/funding/funding-opportunities/funding-programmes-and-open-calls/horizon-europe_en | Cluster-level browsing |
| European Research Council (ERC) | https://erc.europa.eu/apply-grant/open-calls | ERC Starting, Consolidator, Advanced, Synergy grants |
| European Innovation Council (EIC) | https://eic.ec.europa.eu/eic-funding-opportunities_en | EIC Pathfinder, Transition, Accelerator |
| LIFE Programme | https://cinea.ec.europa.eu/programmes/life_en | Environment, climate, biodiversity |
| Interreg / ESIF | https://www.interregeurope.eu/find-a-project | Cross-border regional programmes |
| Eurostars (Eureka) | https://www.eurekanetwork.org/eurostars | SME-focused international R&D |

## Aggregators / Alert Services

| Service | URL | Notes |
|---|---|---|
| GrantsFinder | https://www.grantsfinder.eu | Aggregates EU calls with summaries |
| Subsdy | https://www.subsdy.com | EU grant search with profile matching |
| Funding Box | https://fundingbox.com | Startup/SME-oriented but covers Horizon |
| EURAXESS | https://euraxess.ec.europa.eu | Researcher-focused; covers calls and jobs |
| Research Professional | https://www.researchprofessional.com | Subscription-based; comprehensive alert system |
